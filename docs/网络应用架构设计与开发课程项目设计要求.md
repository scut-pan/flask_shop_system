# 实验题目：

## 设计并实现一个在线购物网站

- 根据所列的**基本功能**完成网站的设计与开发
  - 数据库推荐使用`MySQL`
- 将网站部署上线
  - 以所列的**基本功能**是否实现来考察实验的完成度
  - 占实验成绩的30%
- 代码提交
  - 占实验成绩的30%
- 提交实验报告

# 基本功能要求

- **顾客**
  - 顾客的 **注册**，**登录**，**注销**
  - 展示产品列表
  - 购买流程（浏览/查询 -> 添加至购物车 -> 付款 -> 发送电子邮件确认发货）
  - 查看订单状态和历史
- **销售管理**
  - 商品目录的 **管理**（包括最基本的添加，删除，修改等操作）
  - 订单管理、以及销售统计报表
